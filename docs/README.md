# Documentation

This repository is both a working architecture-governance repository and a
reusable method for agent-driven shared architecture.

The documentation is split the same way successful reusable-method repositories
like `obra/superpowers` do it:

- the root `README.md` explains the purpose quickly
- focused guides live under `docs/`
- adoption guidance is separate from method internals
- reference material is separated from onboarding
